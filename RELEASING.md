# One-time setup

1. Create the repo on GitHub:
   - Go to https://github.com/new
   - Owner: `Spoofch9809`, name: `cookierun-auto-menu-bot`
   - **Public**, no README/gitignore/license (we already have files) -- just
     click Create repository.

2. Push this project to it (run from this folder):

   ```
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/Spoofch9809/cookierun-auto-menu-bot.git
   git push -u origin main
   ```

   The first `git push` will prompt a browser login the first time -- sign
   in as `Spoofch9809` and approve it.

3. Cut the first release (see "Every future release" below) so the update
   checker and the download link in the README have something to find.

# Every future release (after code changes)

1. Bump the version so the update checker and title bar reflect it -- open
   `cookierun_bot.py` and change:

   ```python
   APP_VERSION = "1.0.0"   # -> e.g. "1.1.0"
   ```

2. Double-click `build.bat`. It rebuilds `CookieRunAutoMenuBot.exe` and
   writes `CookieRunAutoMenuBot-vX.Y.Z.zip` (bundling the exe +
   `config.json` + `templates/`) in this folder.

3. Commit and tag the source change:

   ```
   git add -A
   git commit -m "vX.Y.Z: <what changed>"
   git tag vX.Y.Z
   git push
   git push --tags
   ```

4. Publish the GitHub release:
   - Go to https://github.com/Spoofch9809/cookierun-auto-menu-bot/releases/new
   - Tag: pick the `vX.Y.Z` tag you just pushed
   - Title: `vX.Y.Z`
   - Notes: brief summary of what changed
   - Attach `CookieRunAutoMenuBot-vX.Y.Z.zip` (drag it into the upload box)
   - Click **Publish release**

That's it -- everyone's already-running app will show the update banner
next time they open it (it checks on startup), and the Download button
takes them straight to this release page.

# Notes

- The version tag on GitHub (`vX.Y.Z`) and `APP_VERSION` in
  `cookierun_bot.py` should always match -- the update checker compares
  the GitHub release tag against `APP_VERSION`, not the git commit.
- `CookieRunAutoMenuBot.exe`, `build/`, and the release zip are gitignored
  -- they're build output, not source, and get regenerated by `build.bat`.
  Only the zip attached to a GitHub Release is what friends download.
